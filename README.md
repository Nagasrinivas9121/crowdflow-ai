Built **CrowdFlow AI – Smart Physical Event Experience Platform**

Managing large crowds in sports venues is a major challenge. Long queues, congestion, and lack of real-time coordination can impact attendee experience and safety.

To address this, I developed **CrowdFlow AI**, an intelligent system that predicts crowd density and estimates waiting time using AI-based logic.

🔹 Key Features:
• Crowd congestion prediction
• Queue wait time estimation
• AI assistant for quick guidance
• REST API architecture
• Cloud deployment using Google Cloud Run

🔹 Tech Stack:
Spring Boot • React • Docker • Google Cloud Run • H2 Database

🔹 Impact:
Helps improve safety, reduce waiting time, and enhance the physical event experience for attendees in large-scale sporting venues.
